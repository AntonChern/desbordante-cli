# Desbordante CLI

Part of Desbordante platform avaliable at https://github.com/Desbordante
## Installation
```bash
pipx install desbordante-cli
```
Or
```bash
pipx install git+https://github.com/toadharvard/desbordante-cli
```
## Usage
```bash
desbordante --help
```
